# Before the Tutorial

零基础，应当如何开始学习 Python ？

找到合适的入门书籍与教程：首先把本课程教材读懂，不懂就百度，太难的内容可以跳过，然后找到推荐书目或网上教程进行辅助理解。
做好每次实验：Practice makes perfect.练习对于学好一门语言十分重要，不会就反复练习。
保持兴趣，学习解决问题：兴趣是最好的老师，尽量用简单的方式解决问题，减少时间复杂度。
学会检索： 大部分问题都能在博客中找到解决方案，遇到问题先百度。
